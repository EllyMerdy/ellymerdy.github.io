*   Refresh Rate: The tool updates the game memory values once every second.

## 🛡️ Safety and background information

Game trainers act as memory editors. They read the values stored in your computer RAM while the game runs. This tool does not rewrite game code. It sends instructions to the computer to pause or change specific numbers in the background. 

Windows Defender or other antivirus software might flag this file. This happens because the tool touches the process memory of another program. You must add an exclusion for the tool file if your antivirus blocks it. Follow your antivirus support guide to permit the file if the connection blocks.

## 💡 Troubleshooting steps

If the tool does not work, check these common items:

1. Game compatibility: Ensure you play the official version of Chapter 5.
2. Permissions: Always run the tool as an administrator. It needs permission to look at the game process memory.
3. Overlays: Turn off game overlays such as Steam or Discord. These sometimes cause conflicts when two programs read the screen or input at once.
4. Internet disconnects: The tool needs no constant internet connection after the download. Run it while offline if you face connection issues.
5. Crashes: If the game crashes, close the trainer and reload the game. The trainer changes values in real time and some values cause system errors if set too high.
